# Formally — Fitness + Personal Development Platform (Prototype)

## Summary

Formally is an early-stage fitness + social platform. The goal is a clean experience for programs, progress tracking, and community features.

## What’s implemented

- Authentication + authorization (JWT + role-based access)
- Programs, progress tracking, goals, notifications
- Real-time messaging / group chats (Socket.io)
- Admin panel and multi-language support (EN/TR)

## Tech snapshot

- React (frontend)
- Node.js / Express (backend)
- MongoDB (Mongoose)
- Docker + testing setup

## Notes

The repository is private while the product is under development. More links will be added when it’s ready to launch.
