# TheTripMan (Client)

**Live site:** https://www.thetripman.com

## Summary

Production scheduling microsite for a premium transportation business. Built to be fast, mobile-first, and conversion-focused with a reliable booking flow.

## What I shipped

- One-page booking experience: service selection → schedule → details → confirmation
- Two scheduling modes (embed + custom calendar) to match business needs
- Transactional email notifications (confirmations + calendar invites)
- Admin workflow for managing bookings (access-controlled)

## Tech snapshot

- Next.js 15 + TypeScript
- Tailwind + shadcn/ui
- Postgres (Neon) + Prisma
- Scheduling: Cal.com/Calendly + FullCalendar + Google Calendar
- Resend email • GA4 • Vercel
